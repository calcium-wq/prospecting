# Portfolio demo - Animation 3D medicale

Objectif : produire un exemple montrable rapidement si un prospect demande un portfolio.

## Format

- Duree : 20 a 30 secondes.
- Format final : MP4 1080p.
- Style : medical, propre, premium, fond sombre ou neutre, rendu lisible.
- Texte a l'ecran : minimum, uniquement labels courts.
- Pas de promesse scientifique specifique a une vraie biotech.

## Sujet recommande

Mecanisme d'action generique : molecule therapeutique qui se fixe a un recepteur cellulaire et bloque une cascade inflammatoire.

Pourquoi ce sujet :
- applicable a beaucoup de biotechs ;
- facile a comprendre ;
- montre molecule, recepteur, cellule, signalisation ;
- ne depend pas d'un prospect specifique.

## Storyboard

### Scene 1 - Probleme biologique (0-5s)

Vue macro d'une cellule avec plusieurs signaux inflammatoires qui arrivent vers sa membrane.

Visuel attendu :
- cellule semi-transparente ;
- recepteurs sur membrane ;
- particules rouges/orange representant le signal pathologique.

Texte court :
`Signal inflammatoire active`

### Scene 2 - Cible therapeutique (5-10s)

Zoom sur un recepteur. Une molecule therapeutique arrive et se positionne sur le recepteur.

Visuel attendu :
- camera fluide ;
- recepteur en bleu ;
- molecule en blanc/vert ;
- interaction visible sans surcharge.

Texte court :
`Fixation selective`

### Scene 3 - Blocage du signal (10-18s)

Le signal pathologique ne passe plus. La cascade intracellulaire diminue.

Visuel attendu :
- particules rouges ralenties ou bloquees ;
- signaux internes qui passent de rouge a bleu ;
- sensation de controle et de clarte.

Texte court :
`Cascade reduite`

### Scene 4 - Benefice explique (18-25s)

Retour a une vue plus large de la cellule stabilisee.

Visuel attendu :
- environnement calme ;
- contraste avant/apres ;
- rendu propre, pas trop charge.

Texte court :
`Mecanisme rendu clair`

## Prompt de production pour Claude/Blender

```text
Create a 20-30 second premium medical 3D animation showing a generic mechanism of action:
a therapeutic molecule binds selectively to a cell membrane receptor and reduces an inflammatory signaling cascade.

Style: clean Apple-like medical visualization, dark neutral background, translucent cell membrane, blue receptor, white/green therapeutic molecule, red/orange inflammatory particles.

Scenes:
1. Wide shot of a cell receiving inflammatory signals.
2. Camera zooms to one receptor on the membrane.
3. Therapeutic molecule binds to the receptor.
4. Red inflammatory signal particles are blocked, internal signaling fades from red to blue.
5. Final stabilized cell view.

Requirements:
- 1080p MP4
- smooth camera motion
- readable composition
- no clutter
- minimal labels: "Signal inflammatoire active", "Fixation selective", "Cascade reduite", "Mecanisme rendu clair"
- cinematic but scientifically neutral
```

## Phrase a envoyer si un prospect demande un exemple

```text
Oui, je peux vous montrer un exemple de style.

Le plus pertinent reste de prendre 15 min pour comprendre votre mecanisme exact, puis je peux vous proposer un mini storyboard adapte a votre technologie.
```

## Definition de "suffisant"

Cette demo n'a pas besoin d'etre parfaite. Elle doit seulement prouver :
- que le rendu est propre ;
- que tu peux expliquer un mecanisme complexe ;
- que le prospect peut se projeter sur sa propre technologie.
